## Installation Guide:

### Run server

- unzip server.zip to /server
- open a terminal on /server
- run the following script
```
python -m venv venv &&
source ./venv/bin/activate &&
pip install -r requirements.txt &&
python main.py
```
- the server will be run on http://127.0.0.1:5000

### Run client

- unzip client.zip to /client
- open a terminal on /client
- run the following script
```
python -m venv venv &&
source ./venv/bin/activate &&
pip install -r requirements.txt &&
python main.py
```
- the client will be run on http://127.0.0.1:5001
